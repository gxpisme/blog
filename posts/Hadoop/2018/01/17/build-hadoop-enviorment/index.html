CentOS 搭建Hadoop环境

笔者是使用的阿里云的机器

安装软件

ssh

rsync

java

下载JDK

从http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html 选择下载JDK的最新版本 JDK 8。
　　我选择的是：jdk-8u161-linux-x64.tar.gz

解压&安装

配置环境变量

环境变量配置生效
source ~/.bashrc

然后要告诉系统，我们使用的是sun的JDK，而非OpenJDK

安装Hadoop

下载hadoop

从http://hadoop.apache.org/releases.html 找到一个你想要的版本，下载binary包

配置JAVA_HOME

尝试命令bin/hadoop

将会显示hadoop 脚本的使用文档

demo

input的数据

执行命令

./bin/hadoop jar ./share/hadoop/mapreduce/hadoop-mapreduce-examples-3.0.0.jar grep /tmp/input /tmp/output '100'

查看输出结果，就可以了

cd /tmp/output
